# bigDreamerKonval

大梦想家 Konval 版


# 开发
1. 安装依赖
```
npm i 
```

2. 启动开发服务
```
npm run dev
```

## 技术栈

- 框架: React
- CSS: TailwindCSS less
- UI 组件库: Ant Design
- 绘图工具: react-konva
