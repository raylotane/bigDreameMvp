# bigDreamerKonval

大梦想家 Konval 版


# 开发
1. 安装依赖
```
npm i 
```

2. 启动开发服务
```
npm run dev
```

## 技术栈

- 框架: React
- hooks: [ahooks](https://ahooks.js.org/)
- CSS: [TailwindCSS](https://tailwindcss.com/docs/installation) less
- UI 组件库: [Ant Design](https://ant.design/index-cn/)
- 绘图工具: [react-konva](https://konvajs.org/docs/react/Canvas_Portal.html)
- 类似项目：[wickeditor](https://www.wickeditor.com/#/)
- iflow: [iflow](https://platform.iflow.cn/cli/quickstart)


## 主意事项
- 使用 Function Call，提供给AI的工具，入参需要是一个对象，因为 AI 返回的 arguments 是 JSON 字符串
